About the Workshop

Web Development

Join us for a hands-on workshop on building your first website using essential web technologies: HTML, CSS, and JavaScript. This workshop is designed for beginners and will guide you through creating a responsive and visually appealing website from scratch.

Key Topics Covered:

  • Introduction to HTML and structure of web pages
  • Styling websites with CSS (Layout, Colors, Fonts, and Responsive Design)
  • Basic interactivity using JavaScript (DOM manipulation)
  • Introduction to web development frameworks and libraries
  • Creating your first responsive webpage using modern tools

Who Should Attend?

  • Anyone interested in learning web development
  • Students and professionals looking to expand their skill set
  • Beginners with no prior web development experience

Workshop Details:

  • Date: March 12, 2025
  • Time: 10:00 AM - 4:00 PM
  • Location: PHANTASM'25 Conference Hall
  • Instructor: Jane Smith, Senior Web Developer
  • Prerequisites: No prior web development experience required
